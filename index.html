Nexus Engine is a semantic search engine implemented in Java under MIT license. The engine currently is in development stage and it doesn't have any release. The current development stage is about the creating of the base architect of the software to support modularity, different type of databases, the main operating systems, the main platforms and the main Java versions and to build automation testing system what can aid developers.
